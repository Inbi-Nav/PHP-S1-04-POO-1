# Práctica: Programación Orientada a Objetos en PHP  

**Descripción:**  
Colección de ejercicios prácticos diseñados para reforzar los fundamentos de la **Programación Orientada a Objetos (POO)** en **PHP**.  
A través de tres niveles progresivos, se abordan conceptos como clases, objetos, herencia,  métodos, propiedades y uso de estructuras lógicas dentro de un contexto orientado a objetos.

---

## Tecnologías utilizadas
- **Lenguaje:** PHP  
- **Entorno de ejecución:** XAMPP  
- **Editor recomendado:** Visual Studio Code
